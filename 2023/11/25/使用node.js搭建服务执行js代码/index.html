使用node.js搭建服务执行js代码

要使用 Node.js 搭建一个执行 JavaScript 代码的服务器，你可以使用 Express.js 这样的框架来简化服务器的创建和路由管理。下面是一个简单的例子，使用 Express.js 搭建一个执行 JavaScript 代码的服务器：首先，确保你已经安装了 Node.js 和 npm（Node.js 包管理器）。

1、初始化项目

在Linux环境中：在命令行中，创建一个新文件夹，进入该文件夹，并运行以下命令来初始化项目：

mkdir 文件夹名称
cd 文件夹名称
npm init -y

这将创建一个 package.json 文件。在windows环境下也是同理。

2、安装依赖：

npm install express body-parser

3、创建服务器脚本：

创建一个名为 server.js 的文件，并将以下代码复制到文件中：

const express = require('express');
const bodyParser = require('body-parser');
const app = express();
const port = 3000;

app.use(bodyParser.json());

app.post('/execute', (req, res) => {
    try {
        const { code } = req.body;
        const result = eval(`(${code})`);
        res.json({ result });
    } catch (error) {
        res.status(500).json({ error: error.message });
    }
});

app.listen(port, () => {
    console.log(`Server is running at http://localhost:${port}`);
});

4、运行服务：

在命令行中运行以下命令启动服务器：

node server.js

服务器将在 http://localhost:3000 上运行。

5、 Python 测试脚本

使用 requests 库发送 POST 请求：

import requests

url = 'http://localhost:3000/execute'

code = """
function add(a, b) {
    return a + b;
}

add(3, 7);
"""

data = {'code': code}
response = requests.post(url, json=data)

if response.status_code == 200:
    result = response.json()['result']
    print(f'执行结果: {result}')
else:
    error = response.json()['error']
    print(f'错误信息: {error}')
